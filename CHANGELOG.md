# Changelog

All notable changes to this project will be documented in this file.

## [1.1.0] - 2026-02-18
### Added
- **Managed Convex Backend**: Full migration from `localStorage` to a reactive cloud database.
- **Instant Sync**: Real-time data updates across multiple devices/tabs.
- **Unified Schema**: Type-safe database definition for Labs, Profile, Blog, and Contact Messages.
- **Cloud-Backed Lab Recorder**: Session recordings (rrweb) are now persisted in the cloud, removing browser storage limits.
- **Contact Persistence**: Live storage for visitor messages.

## [1.0.0] - 2026-02-18
### Added
- **Lab Recorder Station**: Integrated tool to capture `rrweb` sessions from external labs (vSphere/ESXi).
- **AI Narration Engine**: On-device and cloud-based AI narration for lab demonstrations.
- **Admin Command Center**: Centralized dashboard for managing portfolio content.
- **Initial Persistence**: Robust `localStorage` data layer with Export/Import functionality.

## [0.8.0] - 2026-02-17
### Changed
- **Enhanced Lab Visuals**: Implemented a timeline-based view for lab steps and media.
- **UI Refinements**: Polished Shadcn/UI components and improved dark mode consistency.
- **Responsive Navigation**: Optimized the single-page scroll experience for mobile devices.

## [0.5.0] - 2026-02-16
### Added
- **Portfolio Sections**: Initial implementation of Hero, About (Skills/Certs), Labs Grid, Blog, and Contact sections.
- **GitHub Integration**: Dynamic fetching of repository data to show live project stats.
- **Framework Setup**: Initialized React + Vite + Tailwind CSS + Shadcn/UI foundation.

## [0.1.0] - 2026-02-15
### Added
- **Project Foundation**: Initial repository structure and CI/CD pipelines.
- **Design System**: Established core typography, color palettes, and animation tokens.
